
# Especialización en Ciencia de Datos – Tecnológico de Costa Rica

Repositorio que documenta mi progreso en el Programa de Especialización en Ciencia de Datos del CETIEC (Tecnológico de Costa Rica).

Este repositorio funciona como portafolio técnico de los proyectos desarrollados durante el programa.

---

## Alcance del Programa

La especialización está estructurada en cinco módulos principales:

1. Matemática para Ciencia de Datos  
2. Aprendizaje Automático  
3. Estadística para Ciencia de Datos  
4. Big Data  
5. Minería de Datos e Inteligencia de Negocios  

El programa integra fundamentos matemáticos, inferencia estadística, aprendizaje automático, procesamiento de datos a gran escala y analítica orientada a negocio.
